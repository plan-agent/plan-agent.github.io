# 手动验证清单

## 环境要求
- Node.js 18+
- 已执行 `npm install`

## 启动项目
```bash
npm run dev
```
访问 http://localhost:3000

## 验证链路

### 1. 首页 /
- [ ] 页面标题为"双创营报名与答疑助手"
- [ ] 显示活动亮点（实战导向、跨学科组队、导师辅导）
- [ ] 显示活动时间 / 地点 / 招募对象
- [ ] 显示四大模块介绍
- [ ] 点击"立即报名"跳转到 /register
- [ ] 点击"常见问题"跳转到 /faq

### 2. 报名页 /register
- [ ] 显示表单字段：姓名、学校、专业、年级、邮箱、角色倾向、技能标签、创意想法、参加期待
- [ ] 不填必填字段直接提交，显示中文错误提示
- [ ] 输入错误邮箱格式，显示"邮箱格式不正确"
- [ ] 填写完整正确信息后提交，显示"报名成功！"
- [ ] 提交成功后表单被清空

### 3. FAQ 页 /faq
- [ ] 显示输入框和快捷问题按钮
- [ ] 点击快捷问题"报名截止时间是什么时候？"，返回预置答案
- [ ] 输入"没有编程基础能参加吗"，返回相关答案
- [ ] 输入任意无意义问题，返回兜底提示

### 4. 管理员页 /admin
- [ ] 显示报名总人数
- [ ] 显示已提交的报名记录列表
- [ ] 列表包含：姓名、学校、专业、角色倾向、提交时间
- [ ] 记录按时间倒序排列

### 5. 路演页 /demo
- [ ] 显示项目解决的问题、产品方案、核心功能
- [ ] 数据看板显示报名总人数
- [ ] 显示角色分布
- [ ] 显示填写创意想法的人数
- [ ] 显示下一步升级方向
