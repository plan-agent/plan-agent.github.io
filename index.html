PLAN Lab (Personal Linked Agent Network Lab)

Shenzhen University, Yuehai Campus

Shenzhen, China

PLAN Lab stands for Personal Linked Agent Network Lab.

The name highlights four core ideas:

  • Personal: human-centered intelligence and personalized agent services.
  • Linked: strong interoperability and structured connections among agents.
  • Agent: autonomous, collaborative, and adaptive intelligent entities.
  • Network: large-scale connected ecosystems, or the Agent Internet.

The word PLAN also represents blueprint thinking and long-term system design. Our vision is not isolated agents, but interoperable agent networks that can support next-generation standards, infrastructure, and real-world applications.

PLAN Lab is also known in Chinese as 普兰实验室 / 蓝图实验室.

「普蓝」首期 OPC+Agent 技能培训双创营

主办单位：深圳大学电子与信息工程学院 　｜　 2026/4/20

在人工智能快速发展的今天，AI Agent（智能体）正成为连接技术与商业落地的新桥梁。为了帮助同学们系统理解智能体商业模式、提升从创意到产品的实践能力，特举办「OPC+Agent 技能培训双创营」。

无论你是懂技术的极客，还是擅长社群运营、文案宣传的商科与文科人才，只要你对 AI 创业充满热情，这里都有你的舞台。带着点子来，我们将教你用前沿的 Vibe Coding 方法，在周末两天内打造出能路演的 MVP（最小可行性产品）！

本次双创营为期 2 天（本周六、周日），包含四大核心模块：

课程模块 核心内容 学习目标
模块一：商业逻辑 LLM 与 OPC 商业逻辑、Agent 应用场景 探讨智能体商业模式、MVP 低成本验证与获客策略
模块二：平台应用 “小龙虾”原理与应用、Skills 与 Memory 研发 熟悉平台核心功能，掌握智能体上应用的基础开发能力
模块三：前沿开发 Vibe Coding 方法论、工具链等 学习 SDD/TDD 基础，Claude Code/Codex 开发技术
模块四：实战演练 真实需求分析、方案设计与 Demo 开发 独立完成 PRD 撰写，跑通「从点子到产品」的全过程

本次双创营由电子与信息工程学院资深教师团队倾情授课与指导：张胜利、王滔滔、吴晓晓、童景文、刘芳 等老师将全程陪伴，提供从技术底层到商业逻辑的全方位指导。

招募对象： 全校在读学生（不限专业，技术开发、社群运营、宣传策划等人才均可报名），需自带笔记本电脑参与实训，课程全程免费。
