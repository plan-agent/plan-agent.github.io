# 表单配置指南

## 🎯 使用金数据（推荐，最简单）

### 第一步：创建金数据表单

1. **注册/登录金数据**
   - 访问：https://jinshuju.net
   - 使用微信扫码登录或邮箱注册

2. **创建新表单**
   - 点击"创建表单"
   - 选择"空白表单"
   - 表单标题：`普蓝 OPC+Agent 双创营报名`

3. **添加字段**（按顺序添加）

   | 字段名称 | 字段类型 | 是否必填 |
   |---------|---------|---------|
   | 姓名 | 单行文字 | ✅ 是 |
   | 学号 | 单行文字 | ✅ 是 |
   | 学院 | 下拉框 | ✅ 是 |
   | 专业 | 单行文字 | ✅ 是 |
   | 年级 | 下拉框 | ✅ 是 |
   | 是否自带笔记本电脑 | 单选 | ✅ 是 |
   | 微信号 | 单行文字 | ✅ 是 |

4. **学院选项**（在下拉框中添加）：
   - 计算机与软件学院
   - 电子与信息工程学院
   - 机电与控制工程学院
   - 数学与统计学院
   - 物理与光电工程学院
   - 其他

5. **年级选项**（在下拉框中添加）：
   - 2023级（大一）
   - 2024级（大二）
   - 2025级（大三）
   - 2026级（大四）
   - 研究生
   - 博士生

6. **是否自带笔记本电脑选项**（单选）：
   - 是
   - 否（需要借用）

### 第二步：获取表单链接

1. 点击右上角"发布"按钮
2. 选择"分享"标签
3. 复制"表单链接"（格式：`https://jinshuju.net/f/xxxxxx`）

### 第三步：配置到网页

打开 `register.html` 文件，找到大约第 705 行：

```javascript
const FORM_CONFIG = {
    type: 'jinshuju',
    url: 'YOUR_JINSHUJU_URL',  // ← 替换这里
};
```

将 `YOUR_JINSHUJU_URL` 替换为您的金数据表单链接：

```javascript
const FORM_CONFIG = {
    type: 'jinshuju',
    url: 'https://jinshuju.net/f/xxxxxx',  // ← 您的实际链接
};
```

### 第四步：提交更新

```bash
git add register.html
git commit -m "Configure Jinshuju form integration"
git push origin main
```

等待 1-2 分钟，访问 https://plan-agent.github.io/register.html 测试！

### 第五步：查看数据

1. 登录金数据
2. 点击您的表单
3. 进入"数据"标签
4. 可以查看、导出、分析所有报名数据

## 📊 使用腾讯问卷（备选方案）

### 创建腾讯问卷

1. 访问：https://wj.qq.com
2. 使用微信/QQ登录
3. 创建新问卷，添加相同的 7 个字段
4. 发布后获取问卷链接

### 配置方式

修改 `register.html`：

```javascript
const FORM_CONFIG = {
    type: 'tencent',
    tencent_url: 'https://wj.qq.com/s2/xxxxxxxx',  // 您的腾讯问卷链接
};
```

**注意**：腾讯问卷方式会在新窗口打开问卷，用户需要手动填写。
